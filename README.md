## Глобальные роли

Для разграничения доступа к административным функциям используются глобальные роли:
- `super_admin` — полный доступ
- `admin` — управление пользователями и контентом
- `moderator` — модерация проектов и отзывов
- `support` — техподдержка (только чтение)

> Подробная матрица прав доступна в [Swagger документации](https://localhost:5000/docs)`.
